# Premium DCF Calculator

A comprehensive Discounted Cash Flow (DCF) calculator application for financial analysis and valuation.

## Overview

The Premium DCF Calculator is a powerful tool designed to help investors and financial analysts perform detailed DCF valuations. It provides a streamlined interface for calculating intrinsic value [...]

## Features

- **Cash Flow Projection**: Input and manage projected free cash flows
- **Discount Rate Calculation**: Configure WACC (Weighted Average Cost of Capital)
- **Terminal Value Computation**: Calculate terminal value using perpetuity growth method
- **Valuation Analysis**: Comprehensive valuation metrics and analysis
- **Professional Reporting**: Generate detailed valuation reports

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Amresh01-dev/premium-dcf-calculator.git
cd premium-dcf-calculator
```

2. Install dependencies:
```bash
npm install
```

3. Start the application:
```bash
npm start
```
